# Проєкт "Фортепіано"

У цьому проєкті ми створюємо гру "Фортепіано", де можна грати на віртуальній фортепіанній клавіатурі за допомогою клавіатури комп'ютера або миші. Окрім клавіш у грі також є меню з налаштуваннями інтерфейсу гри.

## Структура проєкта

+ **assets/** - директорія, в якій зберігається все, що потрібно у грі, але не є кодом:
  + **sounds/** - звуки нот
  + **images/** - зображення для інтерфейсу (*іконки, кнопки*)
  + **data/** - файли з даними (*налаштування для користувача, власні звуки, збережені параметри гри тощо*)
+ **ui/** - директорія з файлами створення графічного інтерфейсу:
  + **settings_meny.py** - файл, який описує додаткове вікно налаштувань користувача
  + **slider.py** - файл з описом основного компонента вікна налаштувань - віджета Slider
+ **main.py** - головний файл гри
+ **settings.py** - налаштування гри (*розмір вікна, кольори, список нот*)
+ **sounds.py** - робота зі звуками (завантаження з `assets/sounds/`)
+ **keys.py** - клавіші піаніно та їх анімація
+ **buttons.py** - кнопки меню
+ **effects.py** - ефекти для клавіш (*наприклад, підсвічування при їх натисканні*)
